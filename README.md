# ChatServer

# 编译要求

需要GCC/G++版本在7.0以上，需要先安装boost库，再安装muduo网络库

为了实现离线消息订阅的功能，必须安装redis

为了实现负载均衡，需要安装nginx

且需要手动配置MySQL，实现报告里面的所有表项

环境搭建好后便可以进入ChatServer文件夹，依次执行命令

```powershell
cmake .
make
```

生成的可执行文件在bin目录

```powershell
./ChatServer 127.0.0.1 6000 # 开启服务器
./ChatClient 127.0.0.1 6000 # 在另一终端开启客户端
```
